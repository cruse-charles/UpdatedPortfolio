Let's Review - What is useState?

useState is a built-in React Hook that allows you to add state to functional components. Essentially, it allows your component to "remember" values between renders. You call useState with the initial value of the state. It returns a pair: the current state value and a function to update it. This function updates the state and causes the component to re-render:

How to update our state variables

A direct way to update state variables can be seen below, but it’s not the best way to update state variables that might have some complexity.

setState updates are asynchronous. This means that JavaScript doesn't immediately update the current state and re-render the component when setState is called. Instead, it schedules these operations for future execution and proceeds to the next line of code. This future execution can lead to errors when a subsequent state update depends on the change just scheduled.

So, how do we update state?

Well, we can update state based on the previous state. By passing a function to setState, we ensure accurate updates even with asynchronous modifications. This function receives the previous state value, performs the necessary state update operations, and returns the new state value. This is called the functional method.
